FPGA MEZZANINE CARD (FMC)

An FMC is a daughter card for carrier boards containing a field-programmable gate array (FPGA). Standardizes form factors, connectors, I/O and power interfaces enable modular electronic designs.

2017-30-01 | Single Width FMC Template (Commercial grade)

This project contains several variants of single width modules in commercial grade form factor. All of them are designed for air cooling with either low-pin count (LPC) or high-pin count connector (HPC). Repository: Single Width FMC Template (Commercial grade)

2017-30-01 | Single Width FMC Template (Ruggedized)

This project contains several variants of single width modules in ruggedized form factor. All of them are designed for air cooling or conduction cooling with either low-pin count (LPC) or high-pin count connector (HPC). Repository: Single Width FMC Template (Ruggedized)
