FPGA MEZZANINE CARD (FMC)

An FMC is a daughter card for carrier boards containing a field-programmable gate array (FPGA). Standardized form factors, connectors, I/O and power interfaces enable modular electronic designs.


2017-June-xx | Single Width FMC Template (Ruggedized)

This project contains several variants of single width modules in ruggedized form factor. All of them are designed for air cooling or conduction cooling with either low-pin count (LPC) or high-pin count connector (HPC).



2017-May-30 | Single Width FMC Template (Commercial grade)

This project contains several variants of single width modules in commercial grade form factor. All of them are designed for air cooling with either low-pin count (LPC) or high-pin count connector (HPC is coming soon).



2017-May-01 | ANSI/VITA 57 library for Altium Designer

This project is a schematic and footprint library for printed circuit boards (PCBs) to be designed with Altium Designer (Version 17). Various components for a professional PCB design according to ANSI/VITA 57 FMC standard, such as connectors, mounting holes, and thermal interfaces are included.
